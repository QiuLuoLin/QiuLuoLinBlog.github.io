初识L298N模块

分类：
标签： 模块

日期：2025-05-27 12:14:47

L298N电机驱动模块（目前尚未正确学习，大概率理解有误）

①处：out1和out2，电源输出端。用于给电机供电。

②处：电源输入端（5v~46v）为模块提供电源。

③处：GND。

④处：电源输出端（5V）为编码器或其他模块供电。

其右侧的排针分别为ENA，IN1，IN2，IN3，IN4，ENB

⑤处：ENA

​ 短接（盖上跳线帽 或者 用母-母杜邦线上下相接）会使电机以最大速度工作；

​ 不短接（上下不相接），电机不工作。

⑥处：IN1和IN2，用于控制电机速度和方向。

** 核心信号功能**

信号引脚 作用
ENA 使能信号（PWM输入），控制电机是否工作和调速（高电平有效）。
IA1 方向控制信号1（逻辑电平），与 IA2 组合决定电机转向。
IA2 方向控制信号2（逻辑电平），与 IA1 组合决定电机转向。

控制逻辑真值表

ENA（PWM） IA1 IA2 电机状态 等效电路动作
HIGH/ PWM 1 0 正转 OUT1=VS（电源），OUT2=GND
HIGH/ PWM 0 1 反转 OUT1=GND，OUT2=VS
HIGH/ PWM 1 1 快速刹车 OUT1=OUT2=VS（短接电机）
HIGH/ PWM 0 0 惯性停止 OUT1=OUT2=高阻态
LOW X X 停止 无输出（电机断电）

注：

  • X表示任意状态（0或1）。
  • 刹车 vs 停止：
    • 刹车：电机两端短接，快速制动（耗能）。
    • 停止：电机悬空，惯性滑行。
